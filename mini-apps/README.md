# Mini Apps - Disciplina Desenvolvimento Mobile

Esta pasta contém os mini apps desenvolvidos ao longo da disciplina. Cada mini app está organizado em sua própria pasta, com todos os arquivos necessários para rodar no Android Studio.

## Mini Apps disponíveis

- **calculadora-imposto/**  
  Calculadora de imposto de renda mensal. Permite inserir salário, número de dependentes e gastos com saúde e educação para calcular o imposto devido.

> Cada mini app possui seu próprio README detalhando como rodar e utilizar.
